# scores-table

> Table formatter for scores-parser.

## Install 

```
$ npm install -g scores-table
```

## CLI

#### all results

```
$ scores-table
```

#### live results

```
$ scores-table -l
```

#### finished results

```
$ scores-table -f
```

#### team results

```
$ scores-table -t <team>
```
